# requirements_colab.txt
#
# Runtime dependencies for running this pipeline on Google Colab.
#
# Why this file is separate from requirements.txt:
#   requirements.txt is a `pip freeze` of the local Windows development
#   environment. It contains (a) Windows-only packages such as pywinpty,
#   which cannot compile on Colab's Linux runtime, and (b) local Jupyter and
#   IDE tooling irrelevant to training. It also pins exact versions, which
#   forces pip to re-download packages Colab already provides.
#
# What this file contains:
#   Only the packages actually imported by src/. Versions use >= minimums
#   rather than exact pins, so pip keeps Colab's preinstalled versions
#   whenever they already satisfy the constraint — no redundant downloads.
#
# Deliberately NOT listed: torch and torchaudio.
#   Colab ships a PyTorch build already compiled against its GPU driver and
#   CUDA runtime. Installing a pinned torch here would pull several hundred
#   MB of NVIDIA CUDA libraries and risk replacing a working GPU build with
#   a mismatched one. (torchaudio is additionally unused — all audio I/O in
#   this project goes through librosa and soundfile.)
#
# Usage:  pip install -r requirements_colab.txt
